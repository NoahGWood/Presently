Website Build Deliverable 1

Presently

Main Topic

The main topic of this website is to provide people with a free tool to convert "boring" and difficult to read text into a fun slideshow presentation using artificial intelligence and text to speech software

Main Objectives
  • Provide a means for users to enter their text
  • Allow users to provide a DALL-E backend to generate images from their text
  • Convert text into image based slideshow
  • Narrate text and switch between slides
  • Allow users to save narrated slideshow as a mp4
  • Allow users to export the slideshow as a powerpoint file
  • Do this all client side because life isn't difficult enough
Target Audience

The target audience for Presently is:

  • Students
  • Teachers
  • Dyslexics
  • Visually Impaired
  • Employees who need to make a lot of slideshows
  • Those who have difficulty reading long texts
  • Lazy people like me!

The target audience was determined based on who seemed most likely to want or need to convert a text document into an audio/visual project, but lacks the skills or the time to do so using traditional software solutions like powerpoint. It also seems likely that this will be useful for business uses, as one can simply write about a given topic and have it quickly converted into a slideshow/video presentation without much effort.

Content
  • Content for this site will be generated based on user input using a DALL-E backend. Dall-E was chosen specifically due to its' ability to synthesize new artwork
  • Graphical icons are going to be provided by bootstrap's open-source library because they are free and have a uniform design to them.
  • The text to speech software I will be using for reading out the script is Google's TTS API because I find it to be one of the more natural sounding text-to-speech tools out there, and it is designed to work in the browser client-side as well which will make implementing it easier.
